Plano de Projeto Kanban e Canvas.

Desenvolvi um plano de projeto para otimizar as operações de uma lanchonete, integrando os princípios do Kanban para uma gestão visual eficiente e o Canvas para uma representação gráfica abrangente do modelo de negócios.
